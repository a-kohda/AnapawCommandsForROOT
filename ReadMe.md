# ANAPAW Commands for ROOT
- まだテスト中です
## 使い方
- ls()
  - ヒストグラムの一覧を表示。表示対象は「.ls」で表示されるもの。HID(ヒストグラムID)、次元、タイトルを表示。

- ht(hid)
  - hidを数値指定して、そのヒストグラムを表示する。第二引数でDraw optionの指定可。

- hn() / hb()
  - 現在表示中のヒストグラムの次の(/前の)ヒストグラムを表示する。引数でDraw optionの指定可。

- lgx() / lgy() / lgz()
  - x / y / z 軸をLogスケールにする

- lnx() / lny() / lnz()
  - x / y / z 軸をリニアスケールにする
